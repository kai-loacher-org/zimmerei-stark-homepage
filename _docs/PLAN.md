## 4. Empfohlene Technologie-Stack für Relaunch

### Option A: Statisch (empfohlen für Performance)
- **Framework:** Next.js oder Astro
- **Styling:** Tailwind CSS
- **CMS:** Strapi / Directus / Sanity (Headless)
- **Hosting:** Vercel / Netlify
- **Formulare:** Formspree / Netlify Forms

### Option B: WordPress (einfacher für Redakteure)
- **Theme:** Custom Theme mit Elementor Pro
- **Hosting:** WP Engine / All-Inkl
- **Formulare:** WPForms / Contact Form 7
